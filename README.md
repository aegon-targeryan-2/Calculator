# Calculator App 

A simple desktop calculator application built using Tkinter and CustomTkinter that can perform basic mathematical operations easily.

## Features

1. Addition  
2. Subtraction
3. Multiplication
4. Division
5. Power Operation
6. Square Root Calculation
7. Modulus Operation
8. Simple and Easy-to-Use Interface  

## Installation

1. Download or clone this repository

```bash
git clone https://github.com/Codilosopher/Calculator.git
```

2. Move into the project folder

```bash
cd Calculator
```

3. Run the calculator

```bash
python calc.py
```

## Requirements

- Python 3.x
- Tkinter
- CustomTkinter

```bash
pip install customtkinter
```
